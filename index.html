How to Play

1. Press the START button. QuickClick will flash the first signal. Repeat the signal by clicking the same button.
2. QuickClick will repeat the first signal, then add one. Repeat these two signals by pressing the same buttons in order.
3. QuickClick will repeat these first two signals and add one.
4. Continue playing as long as you can repeat each sequence of signals correctly.
5. If you fail to repeat a sequence exactly, a sound responds with a sound. This means you've lost.
